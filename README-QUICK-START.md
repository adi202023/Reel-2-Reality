# 🚀 Quick Start Guide - Reel to Reality

## Installation & Setup

### 1. Install Dependencies
```bash
# Use legacy peer deps to resolve React version conflicts
npm install --legacy-peer-deps
```

### 2. Start Development Server
```bash
# Start Expo development server
npm start

# Then press 'w' to open web version
# Or directly start web:
npm run web
```

### 3. Access Your App
- **Web**: http://localhost:8081
- **Mobile**: Scan QR code with Expo Go app

## 🎬 Demo for Judges

1. **Welcome Screen** → Experience cinematic animations
2. **Click "Try Demo"** → Instant access (no signup needed)
3. **Explore Dashboard** → View stats, challenges, features
4. **Test Authentication** → Try login/signup forms

## 🛠️ Troubleshooting

### If you see "Cannot find module 'nativewind/babel'":
- Run: `npm install --legacy-peer-deps`
- Restart the server: `npm start`

### If you see React version conflicts:
- The package.json has been updated to use React 18
- Run: `npm install --legacy-peer-deps`

### If you see a blank page:
- Make sure you're accessing the web version at http://localhost:8081
- Check the browser console for any errors
- Try refreshing the page
