# Reel to Reality - Social Challenge App

A React Native Expo app that turns challenges into adventures! Perfect for hackathons and pitches.

## 🚀 Features

- **Beautiful Welcome Screen** with gradient design and smooth animations
- **Challenge Feed** with scrollable cards showing sponsored challenges
- **Challenge Details** with comprehensive information and action buttons
- **Friend Selection** to send challenges to friends
- **Camera/Recording Screen** with gallery integration for proof submission
- **Confirmation Screens** with celebration animations
- **Local State Management** simulating backend functionality
- **Modern UI/UX** with gradients, shadows, and smooth transitions

## 📱 Screenshots & Demo Flow

1. **Welcome Screen**: Gradient background with app logo and "Get Started" button
2. **Challenge Feed**: Scrollable list of challenges with points, difficulty, and sponsor info
3. **Challenge Details**: Full challenge description with "Send to Friend" and "Record Proof" options
4. **Friend Selection**: Choose from a list of mock friends to challenge
5. **Camera Screen**: Professional camera interface with gallery integration
6. **Confirmation**: Celebration screen with points earned and success animations

## 🛠 Tech Stack

- **React Native** with Expo SDK 49
- **React Navigation** for smooth screen transitions
- **Expo Linear Gradient** for beautiful gradients
- **Expo Image Picker** for camera/gallery functionality
- **React Context** for state management
- **Ionicons** for consistent iconography

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd Reel-to-Reality
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Run on device/simulator**
   - Scan QR code with Expo Go app (iOS/Android)
   - Press `i` for iOS simulator
   - Press `a` for Android emulator
